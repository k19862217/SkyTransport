## Transport Web Application
#
This is a transport EC system web application using scala and play framework. 

### Library

The following libs are used.

- Java 1.8
- slick - mysql
- play 2.4.x
- jackson
- fabric

### Build

Please use the following command to build war.

```
$ activator war
```
