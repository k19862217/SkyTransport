## Transport Web Application

### Librares and framework

This is a scala web application using play framework. 

The following libs & database  are used.

- Java 1.8
- slick - mysql
- play 2.4.x
- jackson
- fabric

### Build

Please use the following command to build war.

```
activator war
```
